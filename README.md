# Nextar Backoffice Challenge

The challenge rules are available [here.](https://github.com/nextar/backoffice-desafio-fullstack)

## Install

### npm
```sh
npm install
```

## Run

### [docker-compose](https://docs.docker.com/compose/) (Recommended)
```sh
docker compose up -d
```

### NPM
```sh
npm run start:all
```
